# 💳 UPI Transactions Analysis Dashboard using Tableau

## 📌 Project Overview

This project presents an interactive **UPI Transactions Analysis Dashboard** built using **Tableau**. The dashboard provides meaningful insights into digital payment transactions by analyzing transaction amounts, banking activity, customer demographics, and geographical distribution. Multiple Excel datasets were integrated using Tableau's **Union** feature to demonstrate data preparation and dashboard development.

The project showcases business intelligence techniques through interactive visualizations that help users monitor payment trends and make data-driven decisions.

---

## 🎯 Objectives

- Analyze UPI transaction performance.
- Monitor successful and failed transactions.
- Compare banking activity.
- Understand customer demographics.
- Analyze city-wise transaction distribution.
